# AI Job Search Assistant

## Project Overview
AI Job Search Assistant is a simple web application that helps users search for job opportunities quickly.  
The system fetches real-time job listings using a job search API and displays them in an easy-to-use interface.

## Features
- Search jobs by keyword
- Filter jobs by location
- Display job details such as company, location, and salary
- Simple and interactive UI built using Streamlit

## Technologies Used
- Python
- Streamlit
- Requests
- JSearch API (RapidAPI)
